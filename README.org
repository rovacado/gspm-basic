* GSPM-BASIC
This is a super simple template for the [[https://pypi.org/project/gspm/][gspm]] CLI tool for godot.

* How to Use
first clone the repo
: git clone https://github.com/rovacado/gspm-basic.git
cd into the directory and edit project.yml as needed, then run the install command
: gspm install
finally run the edit command to launch godot straight into the project
: gspm edit
